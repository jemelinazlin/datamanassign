Web 2.0

Category: Web Evolution

Web 2.0 refers to the second generation of the World Wide Web, where websites became interactive and allowed users to create, share, and collaborate on content.

Unlike Web 1.0, which was static and read-only, Web 2.0 introduced platforms such as social media, blogs, wikis, and video sharing sites where users actively participate.

This evolution enabled communication between users and systems, powered by technologies such as HTTP, HTML, and dynamic web applications.

Examples of Web 2.0 platforms include Facebook, YouTube, and Wikipedia, where users contribute content and interact with each other.
