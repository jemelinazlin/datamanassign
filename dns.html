DNS

Category: Internet Infrastructure

The Domain Name System (DNS) is a system used to translate human-readable domain names such as www.google.com into numerical IP addresses that computers use to identify each other on a network.

When a user types a URL into a web browser, the DNS system searches for the correct web server by converting the domain name into its matching IP address. This allows the browser to connect to the correct location on the internet.
