HyperText Transfer Protocol (HTTP)

Category: Web Communication Protocol

HTTP (HyperText Transfer Protocol) is a communication protocol used to transfer data between a web browser and a web server. It defines how requests and responses are structured on the World Wide Web.

When a user opens a website, the browser sends an HTTP request to the web server, which then responds with the requested web page. This process allows users to access content on the World Wide Web.

HTTP operates on top of TCP/IP, which ensures that data is delivered reliably across networks.
